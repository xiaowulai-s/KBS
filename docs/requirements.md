# 项目需求文档 v2.0

## 概述

本项目是一个基于 GitHub Pages 的静态知识库应用，旨在将本地文件整理成结构化的在线知识库。

## 核心功能

### 1. 文件索引管理
- 手动添加/编辑/删除文件条目
- 支持分类和标签体系
- JSON 格式数据存储

### 2. 知识库展示
- 卡片式网格布局
- 响应式设计，适配移动端
- 支持分类和标签筛选

### 3. 全文搜索
- 使用 MiniSearch 构建倒排索引
- 支持中文分词
- 实时搜索结果反馈

### 4. 文件预览
- Markdown 文件实时渲染
- 支持代码高亮
- 图片预览预留接口

### 5. 管理面板
- 表格视图管理所有条目
- 支持 JSON 导入/导出
- 批量操作预留

## 技术栈

| 组件 | 技术 | 版本 |
|------|------|------|
| 前端 | Vanilla JS | ES2020+ |
| 搜索 | MiniSearch | 7.x |
| Markdown | marked.js | 15.x |
| 部署 | GitHub Pages | - |

## 非功能需求

- **性能**：首屏加载时间 < 2s
- **兼容性**：Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **无障碍**：WCAG 2.1 AA 标准
